# spotify-clone
This is fully responsive clone web page of the popular app spotify; HTML, CSS and JavaScript are used to make this web-page ; it dynmically scan the song folder and create the album cards and load the songs inside those albums; the next and previous buttons can be used to change the song also there is a next and previous button for album switching.
we can also use seekbar to forward or backward the playing song; it also support mute and unmute from same volume level.
it perfectly shows the current time and duration of the current playing song
